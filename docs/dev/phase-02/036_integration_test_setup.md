# Task 036: Integration Test Setup

## Goal
統合テストのための環境とヘルパーを準備する

## Tasks

### 036-1: RED - 統合テスト環境のテストを書く
- [ ] テスト用の一時ディレクトリが作成されることを確認するテストを作成
- [ ] テスト後にクリーンアップされることを確認するテストを作成
- [ ] モック環境が正しく設定されることを確認するテストを作成
- [ ] テストを実行して失敗することを確認

### 036-2: GREEN - テスト環境を実装
- [ ] TestEnvironmentクラスを作成
- [ ] 一時ディレクトリの管理を実装
- [ ] モックキーチェーンを実装
- [ ] テストを実行して成功することを確認

### 036-3: REFACTOR - テスト環境の改善
- [ ] テストフィクスチャの整理
- [ ] ヘルパー関数の追加
- [ ] 並列テストのサポート

## Definition of Done
- 統合テスト環境が整備されている
- テストが独立して実行できる
- テストが全て通っている
